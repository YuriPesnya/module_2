Проект 2. Разведывательный анализ данных

Задача проекта
	
	Сформулировать предположения и гипотезы для дальнейшего построения модели.
	Проверить качество данных и очистить их, если это необходимо.
	Определиться с параметрами для модели.

	Создать предварительный набор данных для дальнейшего создания модели для ML.

Данных очень мало. Приходится сохранять по максимуму. Пропусков много. Только 97 строчек из 395 без пропусков.
Выбросов и ошибок ввода не много. Есть разброс данных, связанный скорее всего с реальной статистикой.
Наиболее важными для построения модели параметрами являются sex, address, Medu, Mjob, failures, 
higher, goout, absences.
Есть положительная корреляция между Fedu и Medu, в дальнейшем нужно обратить на это внимание. 

Этапы работы:
1. Загрузка и обзор набора данных.
2. Удалил строчки с пропусками в целевой переменной
3. Проанализировал числовые данные на выбросы и ошибки.
4. Проверил корреляцию и выбрал параметра для дальнейшего анализа.
5. Удалили дублирующую колонку studytime, granular.
6. Создал временный dataframe, в котором заменил NaN на unknown для анализа влияния на score. 
   Сравнивая распределение с помощью boxplot для определенных значений параметров и 
   пропущенных значений, определил параметры, не влияющие на score и выбирал, чем заменить пропуски.
7. На основе теста Стьюдента подтвердил гипотезы о выбранных для финального набора данных параметрах.
8. В итоге в наборе данных осталось 373 строки (из 395) и 17 столбцов (из 28), без пропусков данных. 
   На следующем этапе можно будет оптимизировать набор данных, прогнав их через модель.
